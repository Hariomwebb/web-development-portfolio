# ClientFlow CRM Frontend

ClientFlow CRM is a responsive frontend dashboard for a service agency or sales team.

It is built as a static HTML, CSS, and JavaScript project, so it can run directly in the browser without a framework or build step.

## What It Includes

- Sidebar navigation
- Revenue metrics
- Deal pipeline filter
- Client account table
- Account health filter
- Search by client name
- Follow-up task list
- Toast-style feedback for actions
- Responsive layout for desktop and mobile screens

## Tech Used

- HTML5
- CSS3
- JavaScript

## Files

```text
forntend/
  index.html    Main page markup
  styles.css    Layout, design system, and responsive styles
  app.js        Dashboard data, filters, search, and UI actions
  README.md     Project notes
```

## How To Run

Open this file in a browser:

```text
forntend/index.html
```

Or run a local server from the repository root:

```bash
cd forntend
npx serve .
```

## Interview Talking Point

This project shows how to build a realistic business dashboard with clean UI structure, reusable data rendering, responsive CSS, and basic user interactions without depending on a heavy framework.
