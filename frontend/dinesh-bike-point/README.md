# Dinesh Bike Point

Dinesh Bike Point is a static website for a local two-wheeler service and repair shop.

The site is built with HTML, CSS, and JavaScript. It presents the shop, services, starting prices, and a simple service booking form.

## What It Includes

- Sticky header with navigation
- Workshop hero section with a canvas visual
- Service highlights
- Starting price list
- Accessible booking form
- Estimated service time message
- Customer review section
- Contact footer
- Responsive layout

## Accessibility Details

- Skip link
- Semantic sections
- Labeled form inputs
- Visible keyboard focus styles
- `aria-live` toast feedback
- Good color contrast
- Mobile-friendly layout

## How To Run

Open `index.html` in a browser, or run a local server:

```bash
cd frontend/dinesh-bike-point
python -m http.server 4176
```

Then open:

```text
http://127.0.0.1:4176
```
